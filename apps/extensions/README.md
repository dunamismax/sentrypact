# SentryPact Browser Extensions

Future home for the SentryPact browser companions.

This directory is a placeholder until the Rails control plane (`apps/web`)
publishes the contracts the extensions need. See the root `BUILD.md` for
the active phase plan.

## Planned coverage

- Chrome, Edge, Firefox, and Safari companions
- Cosmetic filtering on top of the platform DNS / VPN enforcement layer
- DoH detection and SafeSearch enforcement
- YouTube Restricted Mode where the platform allows

The extensions are meant to *complement* native enforcement, not replace it.
A user with only the browser extension is not protected to the SentryPact
standard.
