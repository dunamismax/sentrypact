# SentryPact macOS

Future home for the SentryPact macOS enforcement client.

This directory is a placeholder until the Rails control plane (`apps/web`)
publishes the contracts a native client needs. See the root `BUILD.md` for
the active phase plan.

## Planned stack

- Swift
- Network Extension with local VPN/DNS for category filtering
- Family Controls / Managed Settings where Apple permits on macOS
- Optional configuration profiles for managed deployments
- Shared signed blocklist and configuration artifacts with the iOS client

## Status

Not started.
